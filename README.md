# Mobile-Testing-PaaS

### The application can be accessed at : http://3.12.103.66:3000/

### Instructions
For frontend:
1. Go to mtaas folder
2. To install packages: run command `npm i`
3. run cmd: `npm start`

For backend:
1. Go to mtaas-backend folder
2. To install packages: run command `npm i`
3. run cmd: `node index.js`
